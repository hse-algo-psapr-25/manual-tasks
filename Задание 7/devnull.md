# Задача на взаимную рекурсию (маршруты в треугольнике)

В условиях задачи для каждого варианта указан граф с тремя вершинами. Для решения задачи требуется: 
1. Составить систему из 3 рекуррентных соотношений для предложенного графа.
2. Методом исключения свести систему к одному рекуррентному соотношению.
3. Составить и решить характеристическое уравнение.
4. Вывести формулу общего решения.

## Вариант 8:

```mermaid
graph LR

A((A)) --> B((B))
A --> C((C))
B --> A
B --> C
```
Допустимые маршруты:
- A --> B
- A --> C
- B --> A
- B --> C

Найти формулу расчета количества маршрутов, начинающихся в вершине A и заканчивающихся в вершине C. 

1) Система из рекурретных соотношений
* a<sub>n</sub> - число путей длины *n* из A в А
* b<sub>n</sub> - число путей длины *n* из A в B
* c<sub>n</sub> - число путей длины *n* из A в C

Длина 0:

a<sub>0</sub> = 1,  b<sub>0</sub> = 0, c<sub>0</sub> = 0

Длина 1: 

a<sub>1</sub> = 0,  b<sub>1</sub> = 1, c<sub>1</sub> = 1

Формула рекурсии:

* В А прийти можно только из B => a<sub>n</sub> = b<sub>n-1</sub>
* В B прийти можно только из A => b<sub>n</sub> = a<sub>n-1</sub>
* В C прийти можно только из A и B => c<sub>n</sub> = a<sub>n-1</sub> + b<sub>n-1</sub>


2) Метод исключени:

c<sub>n+1</sub> = a<sub>n</sub> + b<sub>n</sub> => Подставляем выражения a<sub>n</sub> и b<sub>n</sub>

c<sub>n+1</sub> = b<sub>n-1</sub> + a<sub>n-1</sub> => Точно также, как у нас было в c<sub>n</sub> 

c<sub>n</sub> = a<sub>n-1</sub> + b<sub>n-1</sub> 

Значит => c<sub>n+1</sub> = c<sub>n</sub> 


3) Характерное уравнение

c<sub>n+1</sub> - c<sub>n</sub> = 0 

Подставновка: с<sub>n</sub> = r<sup>n</sup> 

r<sup>n+1</sup> - r<sup>n</sup> = 0

r<sup>n</sup> (r - 1) = 0

r - 1 = 0 => r = 1


4) Формула общего решения

Так как C<sub>n+1</sub> = C<sub>n</sub>, значит, что все члены последовательности одинаковы

С n = 0: => c<sub>1</sub> не = c<sub>0</sub> 

С n = 1: => c<sub>2</sub> = C<sub>1</sub>

C<sub>n</sub> = 

$$
\begin{cases}
0, n = 0
\\
1, n >= 1
\end{cases}
$$
